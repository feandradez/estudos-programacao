## 📌 Structs em C // Data: 30/05/2025
> Serve como um "agrupar" para os dados/informações. Facilita o acesso aos dados quando forem em maiores quantidades.

```c
struct Pessoa {
    char nome[50];
    int idade;
};
```

**O que aprendi:**
- struct é como se fosse um "molde" para os dados. depois é só inserir os dados/valores
- ponto `.` acessa o campo diretamente
- seta `->` acessa o campo via ponteiro
