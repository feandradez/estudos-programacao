## 📌 Vetores em C // Data: 02/06/2026
> Estrutura que permite armazenar vaiors valores do mesmo tipo em um único nome

```c
//sem vetor:
float nota 1;
float nota 2;
float nota 3;
//com vetor:
float notas[3];
```
//é como criar um "container" com o nome 'notas' que possui 3 valores.


**O que aprendi:**
- armazena valores do mesmo tipo
- serve como um sistema de agrupamento
- valores podem ser definidos na criação ou após através de listas dinâmicas
