## 📌 Funções  em JS // Data: 18/06/2026
> Execução de códigos com mais praticidade, sem precisar ficar reescrevendo o mesmo código várias vezes.

```js
// sem função
console.log(2 + 3);
console.log(8 + 8);

// com funfção
function somar(a, b){
    return a + b;
}

console.log(somar(2, 3)); // imprime 5
console.log(somar(8, 8)); // imprime 16

// exemplo de função de saudação
function saudacao() {
    console.log("olá, seja bem-vindo");
}

saudacao(); // imprime "olá, seja bem-vindo"
```
> invocar a função: "nome_da_funcao();"

**O que aprendi:**
- sintaxe basica
- como criar uma função
- como invocar a função
- parâmetros/argumentos são opcionais
- arrow function
